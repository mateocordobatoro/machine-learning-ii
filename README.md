# Machine Learning II

Repository containing assignments, notebooks, exercises, and reference materials for the Machine Learning II course.

## Project Structure

- `assignments/`: Course assignments and submission files.
- `notebooks/`: Practice notebooks and class exercises.
- `resources/`: Course materials and reference files.
- `.venv/`: Local Python virtual environment. This directory is not included in the Git repository.

## Development Environment

This project uses Python and common libraries for Data Science, Machine Learning, and Computer Vision.

## Environment Setup

Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
